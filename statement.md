# Project Statement
---
## Problem statement
The increasing need for personal health monitoring combined with limited access to immediate medical consultation creates a gap in healthcare management. Many individuals lack the knowledge to interpret their vital signs or recognize when medical attention is required. This project addresses these challenges by providing an automated system that monitors health parameters and provides instant, understandable health assessments.
---
## Scope of the project
### Included:
- Patient information management system
- Vital signs data entry and storage
- Automated health risk assessment
- Real-time alert generation
- Console-based user interface
- In-memory data persistence

### Excluded:
- Graphical user interface (GUI)
- Database integration
- Multi-user authentication system
- Mobile application development
- Medical device integration
- Clinical diagnosis capabilities
---

## Target users
1. **Individual users** - People wanting to monitor their personal health metrics
2. **Family caregivers** - Individuals responsible for monitoring family members' health
3. **Small clinic staff** - Medical personnel in small healthcare facilities for basic patient tracking
---

## High-level features
1. **Patient Management System**
   - New patient registration with auto-generated IDs
   - Patient information storage (personal details and medical history)
   - Patient lookup and directory services

2. **Health Monitoring Module**
   - Comprehensive vital signs input interface
   - Multiple parameter tracking (5 key health indicators)
   - Historical data maintenance and retrieval

3. **Intelligent Analysis Engine**
   - Multi-parameter health risk assessment
   - Color-coded health status indicators
   - Plain-language health explanations

4. **Alert and Notification System**
   - Threshold-based alert triggering
   - Critical condition identification
   - Active alert management and display

5. **Reporting Capabilities**
   - Individual patient health reports
   - Health trend analysis
   - System usage statistics
